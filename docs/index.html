For authors

Blockchain proof of authorship for independent authors

AuthorChain helps authors prove that a specific manuscript file existed at a specific time. The private file is hashed (SHA-256), and only that hash is registered on Base Sepolia — the manuscript itself never goes on-chain and is never exposed publicly. The result is a tamper-evident, independently verifiable ownership record an author can point to at any time.

The platform

AI + Web3 publishing infrastructure

Beyond proof of authorship, AuthorChain brings the publishing stack together: AI-assisted author tools, Stripe payments with automatic royalty tracking, ISBN-13 validation and EAN-13 barcode generation, and protected buyer access so that only paying readers can download a book. Authors keep private manuscripts private while selling and delivering securely.

Honest status

AuthorChain is currently a validated MVP using testnet blockchain (Base Sepolia), Stripe test mode, local protected storage, and placeholder auth seams. These are intentional MVP choices, each behind an upgrade seam (storage driver, auth module, payments/registry clients).

Live demo app status

The full interactive AuthorChain app cannot run directly on GitHub Pages because it uses server routes, PostgreSQL, Stripe webhooks, and protected file delivery. This page documents the validated demo and links to verifiable proof. A hosted app demo can be deployed next on Vercel or similar infrastructure (with a database, storage, and webhook endpoints).
